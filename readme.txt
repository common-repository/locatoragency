== Description ==

Plugin provides a management system of physical locations, adding a new type of publication 'locator_agency', a LOCATOR shortcode is added to a page to display the map and related information.
The administration has set up a form for the configuration to display the map.

= Features =

1. New custom post type 'locator_agency'.
2. ShortCode 'LOCATOR' to add a page.
3. Configuration form to display the map.

== Installation ==

1. Download the plugin.
2. Upload the LocatorAgency folder to your site's wp-content/plugins directory.
3. Active the plugin in the Wordpress Panel.
4. Start creating new locator.
5. Setting's Page (Setting -> Locator Option )
6. Create a page, then put the shortcode [LOCATOR]
7. Visit the page created
